# Power BI Sales Dashboard

## Project Overview
This project is an interactive sales performance dashboard built using Power BI.  
The dashboard helps analyze overall business performance using key metrics, trends, and filters.

## Dataset
The dataset contains retail sales data with the following fields:
- Order Date
- Customer Name
- Segment
- Region
- Category
- Sub-Category
- Product Name
- Sales
- Quantity
- Discount
- Profit

## Key KPIs
The following business KPIs were created using DAX:
- Total Sales
- Total Profit
- Total Quantity
- Profit Margin (%)

## Dashboard Features
- KPI cards for quick business overview
- Monthly Sales Trend (Line Chart)
- Sales by Category (Bar Chart)
- Top 10 Products by Sales (Table)
- Interactive slicers for:
  - Region
  - Segment

## Tools & Skills Used
- Power BI Desktop
- DAX (Measures)
- Data Modeling
- Data Visualization
- Business Analytics
